Welcome

About NSClient++

NSClient is an agent designed originally to work with Nagios but has since evolved into a fully fledged monitoring agent which can be used with many monitoring tools. If you want to know the latest news and updates see the news section.

  • About NSClient++
  • Latest news

Getting started

I know NSClient++ can seem daunting. But is really isn't, especially not now with all the documentation. But it is "open-ended" so before you start be sure to read through the Getting-started guide to get a step by step introduction and we are sure you will be on a journey towards #MonitoringLove.

Getting involved is easy! If you have questions or want to connect with the community head over to the Forums. Or if you have issues, feature requests you can use our issue tracker on GitHub. We are also interested in all contributions big or small.
